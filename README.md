# Card de Perfil Interativo

Esse projeto foi desenvolvido como atividade prática da disciplina de **Programação Web II**.

O objetivo é criar uma página web com **cards de perfil interativos**, permitindo que o usuário interaja com os elementos visuais utilizando **HTML, CSS e JavaScript puro**.

---

## Objetivo

Construir uma interface com múltiplos perfis de usuários onde seja possível:
- Visualizar informações pessoais
- Interagir com botões dinâmicos
- Alterar o visual da página e dos cards

---

## Funcionalidades

### Estrutura do Card
Cada card de perfil contém:
- Foto/avatar  
- Nome do usuário  
- Cargo/profissão  
- Breve descrição (bio)  
- Lista de habilidades (mínimo de 3)  
- Contador de seguidores e seguindo  

---

### Interações Implementadas

- **Modo Claro/Escuro**
  - Alterna o tema de toda a página  

- **Mudança de cor do card**
  - Alterna entre 3 cores pré-definidas  

- **Seguir / Deixar de Seguir**
  - Alterna o texto do botão entre "Seguir" e "Seguindo"  
  - Atualiza o número de seguidores (+1 / -1)  
  - Muda o estilo do botão  

- **Centralização do card**
  - Ao clicar em um card, ele é destacado no centro da tela  

- **Troca de avatar**
  - Permite alterar a imagem do perfil  

---

## Estilização

- Interface moderna e visualmente agradável  
- Efeitos de **hover** nos botões  
- Layout organizado com **Flexbox** ou **Grid**  
- Design **responsivo** para diferentes telas
